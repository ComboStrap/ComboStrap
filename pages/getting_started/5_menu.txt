====== ComboStrap Getting Started - Creation of a top menu bar======


===== About =====
The [[strap:strap|strap template]] supports the creation of a slot page called [[strap:header_slot|header slot]] in order to create a [[ui:menubar|menu bar]] at the top location of the page.


===== Steps=====

  * Create a page called '':slot_header'' (at the root directory of your pages)
  * Enter the below [[ui:menubar|menu bar]] example



If you have uploaded a icon in the [[4_favicon|Favicon step]], a minimal ''header slot'' with a [[ui:menubar|menu bar]] would look like:

<webcode name="Default">
<code dw>
<menubar breakpoint="md" align="center">
<brand title="My Brand">
<icon name="logo.svg" />
My Brand name
</brand>
<collapse>
<group expand="true">
<dropdown name="Menu">
[[#|item]]
[[#|item]]
</dropdown>
</group>

<btn shadow="true" color="white">[[getting_started:getting_started|Get Started]]</btn>
<search/>
</collapse>
</menubar>
</code>
</webcode>

More ... see the [[ui:menubar|menu bar page]]
