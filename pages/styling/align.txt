====== ComboStrap Styling - Align Attribute ======


===== About =====
''align'' is an [[styling:styling|styling attribute]] of the [[styling:html|HTML superset component]]

It permits to align:
  * a [[#block|block]]
  * or its [[#text|text]]

<note tip align="center" width="fit-content">
If you want more control on the position, you may also use a [[ui:grid|grid]]
</note>

<badge>Fully operational, since [[release:release|version 1.4]]</badge>

===== Type =====
==== Block ====

How to align a [[ui:block|block]]

The ''align'' attribute on a box, supports the values:
  * ''left'' (by default, the block is aligned to the left)
  * ''center'', 
  * ''right'' 

Example:
<webcode name="Centered" >
<code dw>
<note tip align="right"  width="300px">
=== Block to the right ===

As every block by default takes the whole available [[dimension|width]], you need to add a [[dimension|width attribute]] when you want to align it.

</note>
</code>
</webcode>


<note>
You can also [[ui:image#align|align an image by passing the align parameter]]
</note>

==== Text ====

How to align the text in a [[ui:block|block component]]

The ''text-align'' attribute on a block, supports the value:
  * ''center''
  * ''right''
  * ''justify''
  * ''left'' (default)

Example:
<webcode name="Text ALign" >
<code dw>
<card text-align="center" width="300px">
=== Centered Text  ===

A text in the center

</card>
</code>
</webcode>


<note tip>
=== For the geek ===
This is just a CSS shortcut to the [[https://developer.mozilla.org/en-US/docs/Web/CSS/text-align|text-align CSS attribute]]
</note>
