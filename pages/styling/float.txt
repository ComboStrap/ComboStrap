====== ComboStrap Styling - Float ======


===== About =====
''float'' is a [[styling|styling attribute]] that allows content along the side of the ''floated'' component whereas an [[align|aligned component]] does not.


===== Articles Related =====
{{backlinks>.}}

===== Syntax =====

<code dw>
<component float="value">
</code>
where value is a composition of one of the below floated value separated by a space. 

^ [[breakpoint|Breakpoint]] / Position ^ Left ^ Right ^ None ^ Default ^
| ''xs'' | ''xs-left'' | ''xs-right'' | ''xs-none'' | ''xs-none'' |
| ''sm'' | ''sm-left'' | ''sm-right'' | ''sm-none'' | - |
| ''md'' | ''md-left'' | ''md-right'' | ''md-none'' | - |
| ''lg'' | ''lg-left'' | ''lg-right'' | ''lg-none'' | - | 
| ''xl'' | ''xl-left'' | ''xl-right'' | ''xl-none'' | - |
| [[#floatDefaultBreakpoint|default breakpoint]] | ''left'' | ''right'' | ''none'' | - |


===== Example =====

==== Right ====
A value of ''right'' will:
  * float the element from the [[#floatDefaultBreakpoint|default breakpoint]]
  * not float the element on the extra small device because of the ''xs'' device

==== floatDefaultBreakpoint ====

The [[:configuration|configuration]] ''floatDefaultBreakpoint'' defines the screen size that will be used for the floated value: ''left'', ''right'' and ''none''
