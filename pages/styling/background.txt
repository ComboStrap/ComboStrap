====== ComboStrap Styling - Background ======


===== About =====
The ''background'' element manage and add a background to its parent element.

It's mostly used with [[ui:slide|slide component]] to create compelling landing page.


The ''background'' is defined as being the content plus the [[styling:spacing|padding]] (ie not the margin)
===== Articles Related =====
{{backlinks>.}}

===== Syntax =====


<code dw>
<background fill="cover|tile" color="value" opacity="0 to 1" scroll="local|fixed">{{image}}</background>
</code>
where:
  * ''image'' is an [[ui:image|image]]
  * ''fill'' defines how the image takes the space of its container.
      * ''cover'' (default) - The image is scaled within its container and cropped, if necessary (ie responsive background)
      * ''tile'': The image is repeated until the container is filled.
  * ''color'': a [[color|color value]] - if you have set an image, the color value is seen while the image is loading.
  * ''opacity'' to bring depth level on color. By layer, below are typical values.
    * First: 0.15
    * Second: 0.3
    * Third: 0.45
    * Fourth: 0.6
    * Fifth: 0.75
    * Sixth: 0.9
  * ''scroll'': the scroll effect - how the background is moving while scrolling
      * ''local'': the image is moving with its container
      * ''fixed'': the image stays at the same position on the screen


<note>
If you set more than one background, the first background in the list is the farthest on the stack. 
</note>

==== color ====

You can set a background color with:
  * the [[#color attribute|color attribute of the background component]]
  * via the [[#background color attribute|background color attribute]]
  * or any [[https://getbootstrap.com/docs/5.0/utilities/background/|bootstrap background utility class]] in the [[class|class attribute]]

=== Name / Value ===

All [[color#name|color name]] are supported . All ''color'' value described in the [[color|color page are supported]].


=== Gradient ===
[[:start|ComboStrap]] implements also a ''gradient'' color naming scheme in order to show a ''linear color gradient'' in the background.

<badge>Since [[:changes|version 1.2]]</badge>

If you want to have a gradient on your color, just add the ''gradient-'' prefix to a color value.


Example with a [[ui:note|note]]:
<code dw>
<note info background-color="gradient-light" >
...
</note>
</code>




<note important>
Because the gradient generated a image, you can't use a gradient color and an image at the same time in a background element.
You just need to add another background.
</note>

==== CSS Pattern ====

https://bansal.io/pattern-css :
  * based on four css properties linear-gradient, radial-gradient, repeating-linear-gradient and repeating-radial-gradient
  * built with two colors only:
      * transparent: the transparent color allows to see the background color
      * currentColor: the current active color inside a element. 

Example:
<code css>
.pattern-diagonal-stripes-sm {
    background: repeating-linear-gradient(45deg, transparent, transparent 10px, currentColor 10px, currentColor calc(2 * 10px));
}
</code>
