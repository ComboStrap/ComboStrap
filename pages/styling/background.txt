====== ComboStrap Styling - Background ======


===== About =====
The ''background'' element manage and add a background to its parent element.

It's mostly used with [[ui:slide|slice component]] to create compelling landing page.


The ''background'' is defined as being the content plus the [[styling:spacing|padding]] (ie not the margin)
===== Articles Related =====
{{backlinks>.}}

===== Syntax =====


<code dw>
<background sizing="single|tile|cover">{{image}}</background>
</code>
where:
  * ''image'' is an [[ui:image|image]]
  * ''sizing'' defines the main image behavior:
      * ''cover'' (default) Scale the image within its container, crop, if necessary
      * ''single'': Show the image once (Spaces not covered are filled with the background-color)
      * ''tile'': Repeat the image if the container is larger than the image
  * ''color'': a [[color|color value]]
  * ''opacity'' to bring depth level on color. By layer, below are typical values.
    * First: 0.15
    * Second: 0.3
    * Third: 0.45
    * Fourth: 0.6
    * Fifth: 0.75
    * Sixth: 0.9
  * ''height'': ''100vh'' the height of the screen (vh = viewport height)
  * ''z-index''



===== Example =====

==== Color ====
All ''Background color'' value described in the [[color|color page are supported]].


You can set a background color with:
  * the [[#color attribute|color attribute of the background component]]
  * or via the [[#backgroundcolor attribute|background color attribute]]

=== Color attribute ===

<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<note info>
<background color="gradient-light"/>
==== Linear Background Gradient Color ====
A info note with a ''light gradient color'' as background
</note>
</code>
</webcode>

=== Background color attribute ===
The ''background-color'' attribute is just a short cut.
<webcode name="Default" frameborder="0">
<code dw>
<note info background-color="#6FE2E9" >
==== Direct Background  Color ====
A info note with a background color set via the background-color attribute
</note>
</code>
</webcode>


<note>
You can also use the [[https://getbootstrap.com/docs/5.0/utilities/background/|bootstrap background utility class]] in the [[class|class attribute]]
</note>



==== Background Transformation ====


<code dw>
<background width="600px" height="600px" color="#00f1c9" radius="12px"></background>
</code>
<code css>
element.style {
    transform: rotate(45deg) translateY(-100px) translateX(220px);
    position: absolute;
    width: 600px;
    height: 600px;
    background: #00f1c9;
    border-radius: 12px;
    z-index: -1;
}
/** or for coordinate margin-top: -220px; margin-left: 100px; **/
</code>

==== CSS Pattern ====

https://bansal.io/pattern-css :
  * based on four css properties linear-gradient, radial-gradient, repeating-linear-gradient and repeating-radial-gradient
  * built with two colors only:
      * transparent: the transparent color allows to see the background color
      * currentColor: the current active color inside a element. 

Example:
<code css>
.pattern-diagonal-stripes-sm {
    background: repeating-linear-gradient(45deg, transparent, transparent 10px, currentColor 10px, currentColor calc(2 * 10px));
}
</code>
