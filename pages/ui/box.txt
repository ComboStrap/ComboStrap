====== ComboStrap UI - Box ======


===== About =====
A ''box'' is a component that permits to contain other [[ui|components]].

This is useful to:
  * define the elements that should [[collapse|collapse]]
  * or create a [[styling:styling|styled box]].

<note>
On a HTML level, this is just a ''div'' element
</note>
