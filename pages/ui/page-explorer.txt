====== ComboStrap UI - Page Explorer  ======



===== About =====
A ''page-explorer'' is a [[navigation|navigational component]] that creates a navigation pane in the form of:
  * a list 
  * or clickable levels of folder hierarchy


This component is mostly used in a [[strap:side_slot|side slot]] or [[strap:sidekick_slot|sidekick slot]]


===== Articles Related =====
{{backlinks>.}}

===== Syntax =====

<code dw>
<page-explorer type ns="current">
  <page>$path - $title</page>
  <namespace>$path - $title</namespace>
  <home>$path - $title</home>
</page-explorer>
</code>
where:
  * the attributes are:
      * [[styling:type|type]] that defines the layout and may be one of this two value:
          * [[#list|list]] to create list navigation pane (the default)
          * [[#tree|tree]] to create tree navigation pane
      * ''ns'' that defines the ''namespace'' (Default to the ''namespace'' of the requested page)
  * the sub-element are:
    * ''page'' defines a [[template|template]] for each page in the namespace
    * ''home'' defines a [[template|template]] for the home page of the namespace
    * ''namespace'' defines a [[template|template]] for each child namespace of the actual namespace 
    *  ''parent'' defines a [[template|template]] for the parent namespace of the actual namespace (used only in the [[#list|list layout]])
  


<note >
''page-explorer'', ''home'', ''page'', ''parent'' and ''namespace'' are all [[styling:html|superset HTML component]].
</note>

==== List ====

{{:ui:page-explorer-list-name-to-ui-region.png|}}

<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<page-explorer color="blue" height="250px" width="400" scroll="toggle">
    <home>[[$path?underline=false|$title]]</home>
    <page>
        [[$path?underline=false|<icon name="file"/> <pipeline>"$h1" | replace("ComboStrap UI - ","")</pipeline>]]
    </page>
</page-explorer>
</code>
where the ''page'' template is composed of:
  * an [[icon|icon]]
  * a [[ui:link|link]] without [[styling:underline|underline]]
  * and a [[pipeline|pipeline]] to modify the [[semantic:h1|heading 1]] of the page.
</webcode>

==== Collapsible Tree ====


<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >

    <ns><pipeline>"$h1" | cut("-",2) | trim() </pipeline></ns>
    <page>
        [[$path|<icon name="file" color="#0000004d" width="20"/>  <pipeline>"$h1" | cut("-",2) | trim() </pipeline>]]
    </page>
</page-explorer>
</code>
where the ''page'' template is composed of:
  * an [[icon|icon]]
  * a [[ui:link|link]]
  * and a [[pipeline|pipeline]] to modify the [[semantic:h1|heading 1]] of the page.
</webcode>
