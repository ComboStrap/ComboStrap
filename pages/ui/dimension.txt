====== ComboStrap - Sizing (Width and Height) ======


===== About =====
Every [[box|Box components]] and even the whole main page dimension may be resized in [[#width]] and [[#height]].

===== Articles Related =====
{{backlinks>.}}


===== Dimension =====

==== Width ====
=== Box ===

On a [[box|box]], the ''width'' attribute will apply a ''maximum-width'' constraint. When your website will be seen on a screen below this ''width'', the box will adapt accordingly keeping your website responsive.

Example: a [[note|note]] will take all the space by default, if you want to constraint the box to a lower size, apply the ''width'' attributes.

<webcode name="Width" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<note width="200px" align="center">
=== A constrained note ===
I'm a note being constraint in a box with a maximum width of 200 pixels.
</note>
</code>
</webcode>
<note tip>
If you want to center the box, use the [[align|align attributes]]
</note>
=== Page ===
If you want to constraint a whole page, you can use a [[container|container]]

<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<container width="700px">
====== Page title ======
A page that is centered and constraint in width
===== Section =====
Lorem ipsum
</container>
</code>
</webcode>


==== Height ====
The ''height'' attribute is also a [[style|style shortcut]] that will apply a maximum weight.

Example:
Example: a [[note|note]] will take all the space by default, if you want to constraint the box to a lower size, apply the ''width'' attributes.

<webcode name="Width" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<note important width="200px" height="100px" align="center">
=== A constrained note in height ===
I'm a note being constraint in a box with a maximum height of 100 pixel.
</note>
</code>
</webcode>

===== For the geek =====
<note tip width="600px" align="center">
The ''width'' component attribute is a [[style|style shortcut]] and could have been written
<code html>
<note style="max-width:200px">
</code>


The ''height'' component attribute is a [[style|style shortcut]] and could have been written
<code html>
<note style="height:100px;overflow:auto">
</code>
</note>
