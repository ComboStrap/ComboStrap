====== ComboStrap UI - Jumbotron ======


===== About =====
A ''jumbotron'' is a simple [[block|box component]] that emphase prominently its content.

<note>
For a hero unit style content, you can also use a [[slide|slide]].
</note>

<badge>Since [[release:release|version 1.4]]</badge>

This component was removed from [[https://getbootstrap.com/docs/5.0/migration/#jumbotron|Bootstrap version 5]] but we still support it.
===== Articles Related =====
{{backlinks>.}}

===== Syntax =====
<code dw>
<jumbotron>
=== Title ===
Content
</jumbotron>
</code>
