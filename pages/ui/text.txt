====== ComboStrap UI - Text ======


===== About =====
''text'' is a [[typo|typographic]] component that permits to style one or several paragraph at once.

<badge>Since [[release:release|version 1.4]]</badge>
===== Articles Related =====
{{backlinks>.}}

===== Syntax =====

<code dw>
<text type>
First paragraph

Second paragraph
</text>
</code>
where:
  * [[styling:type|type]] can take the values:
      * ''lead'': It makes a paragraph stand out by setting a bigger font-size and weight
      * ''h1 to h6'': when you want to match the font styling of a [[heading|heading]]. <inote>If you want to set a title for a component, use the [[title|title component]] instead</inote>
      * or any class that you created with the [[styling:userstyle|userstyle class]].
  * ''line-spacing'' sets the space between the lines of a paragraph.
    * ''1''
    * ''sm''
    * ''base''
    * ''lg''
  * [[styling:align#text|align]] 

==== Align ====
It supports the value:
  * ''center''
  * ''right''
  * ''justify'' (While, aesthetically, justified text might look more appealing, it does make word-spacing more random and therefore harder to read.)
  * ''left'' (default)

It can also be applied on other component with the [[styling:align#text|text-align attribute]].

https://getbootstrap.com/docs/5.0/utilities/text/#text-alignment

==== Lead =====

Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. 
</text>

Lorem ipsum dolor sit amet, consectetur adipiscing elit, 

sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. 
</code>
