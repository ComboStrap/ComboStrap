====== ComboStrap UI - Card Columns ======



===== About =====
With the ''card-columns'' components, you can put a [[card|card]] or [[ui:blockquote|blockquote]] in a column format.


===== Articles Related =====
{{backlinks>.}}

===== Syntax =====
<code xml>
<card-columns>
   <!-- Any type of card -->
</card-columns>
</code>
===== Example =====


<webcode name="Default" frameborder=0>
<code dw>
