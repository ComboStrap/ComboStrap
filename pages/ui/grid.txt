====== ComboStrap UI - Grid (Row and Column) ======


===== About =====
The ''grid'' component is a composition of two tags:
  * the ''row'' tag
  * and the ''column'' (or ''col'' tag)

<badge>Since [[release:release|version 1.2]]</badge>

===== Articles Related =====
{{backlinks>.}}

===== Syntax =====
<code dw>
<row type>
   <col></col>
   <col></col>
</col>
</code>
where
  * ''type'' may be the value ''auto'' - each column will takes their natural width and never overflow

<note>
They are actually [[styling:html|superset HTML component]], you can therefore use all [[https://getbootstrap.com/docs/5.0/layout/grid/|Grid bootstrap class]]
</note>

<badge>Attributes are [[release:release|in the planning]]</badge>

==== Basic two columns ====

On a grid of 12 columns, a row with two columns:
  * evenly distributed on a [[styling:breakpoint|large desktop (lg)]]
  * stacked on a [[styling:breakpoint|mobile phone]] - col-12 is for the [[styling:breakpoint|extra small breakpoint]].

        First column
    </col>
    <col 
        class="col-12 col-lg-6"
        background-color="blue"
        color="white">
        Second column
     </col>

==== Custom Card ====
You can also create a [[ui:card|card]]-like component based on row and columns.

Example where the whole row:
  * is presented as a card 
  * and is [[styling:clickable|clickable]] 

<webcode name="Default" frameborder="0">
<code dw>
<row clickable="true" background-color="light" class="g-0" spacing="m-3">
  <col class="col-md-6" spacing="p-4">
    {{:styling:subway_card_tile.png?0x200|}}
  </col>
  <col class="col-md-6" spacing="p-4">
    === Card Clickable Grid ===
    Another instance of placeholder content for this other custom component. It is intended to mimic what some real-world content would look like, and we're using it here to give the component a bit of body and size.
    [[#|Go somewhere]]
  </col>
</row>
</code>

</webcode>


===== Configuration =====
==== gridColumns ====
You can change the default number of columns from 12 to 16. See [[strap:dynamic_grid]]
