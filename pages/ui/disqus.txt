====== ComboStrap UI - Disqus Forum / Discussion ======


===== About =====
The ''disqus'' component will add a disqus forum.

<note>
In development
</note>


===== Syntax =====

<code dw>
<disqus 
    shortName="forumShortName"
    id="disqusIdentifier" 
    categoryId="categoryIdentifier"
    url="urlIdentifier''
/>
</code>

^ Attributes  ^ Default to ^ Description ^
| ''shortname'' | the [[#disqusForumShortName]] configuration | It's the forum's shortname. It's the unique identifier for your website as registered on Disqus. |
| ''id'' | the [[semantic:canonical|canonical value]] and if not set to the [[doku>pagename|Dokuwiki Id]] | The discussion ''identifier'' |
| ''url'' | the [[semantic:canonical|canonical URL]] if set | The ''Url'' disqus configuration is used to come back from Disqus to your page |
| ''category'' | Not set | It's the category id number to be used for the current page |
| ''title'' | the [[semantic:title|title]] | The title of the thread |










===== Configuration =====
==== disqusForumShortName ====

The ''disqusForumShortName'' [[:configuration|configuration]] is the unique identifier for your website as registered on Disqus. [[https://help.disqus.com/en/articles/1717084-javascript-configuration-variables|Reference]]
