====== ComboStrap UI - Color ======





===== About =====
Every [[ui|ComboStrap UI component]] can be colorized with one of the following attributes
  * ''color'': The color of the text
  * ''background-color'': The color of the background
  * ''border-color'': The color of the border

<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >

A teal note with a black text and a blue border
</note>

</webcode>

===== Bootstrap Color Set =====

<NOTE important>
For your information, Bootstrap does not assign the same color set by name for a component.
</NOTE>

For instance, you got this colors for a [[button#primary|Primary button]]
<code css>
.btn-primary {
    color: #fff;
    background-color: #075ebb;
    border-color: #075ebb;
}
</code>
and this one for a [[note#primary|primary note]]
<code css>
.alert-primary {
    color: #004085;
    background-color: #cce5ff;
    border-color: #b8daff;
}
</code>
