====== ComboStrap UI - File ======


===== About =====
The ''file'' component:
  * takes over the manipulation of the default [[doku>wiki:syntax#code_blocks|dokuwiki file block]]
  * is a wrapper around the [[styling:prism|Prism]] component to add:
      * syntax highlighting
      * clipboard copy


===== Articles Related =====
{{backlinks>.}}

===== Example =====

You can create block of code like this one:

<file md line-numbers="true">
# Markdown header

The `file` component:

  * takes over the manipulation of the default [dokuwiki file block](http://www.dokuwiki.org/wiki%3Asyntax#code_blocks)
  * is a wrapper around the [Prism component](/prism) to add:
      * syntax highlighting
      * clipboard copy
</file>


===== Syntax =====
This is a [[styling:prism|Prism component]] and follows the [[styling:prism#syntax|prism syntax]]. You should then check the [[styling:prism#syntax|prism syntax documentation]] for all details.







===== Configuration =====

==== Enable / Disable ====
The ''file_enable'' [[:configuration|configuration]] permits to enable / disable this component.


==== Theme ====

This component gets the [[styling:prism|theme]]  chosen

For [[styling:userstyle#class|custom css style]], there is also an extra [[styling:userstyle#class|userstyle class]] created on the ''pre'' element with the ''pref'' suffix. This element contains the code and the button.

Example:
  * Change the background color of [[ui:file|file]]
<code css conf/userstyle.css>
.file-combo-pre{
    background-color: #e9ecef !important;
}
</code>
