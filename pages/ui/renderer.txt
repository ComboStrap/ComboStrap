====== ComboStrap UI - Combo Renderer (Rplus) ======

===== About =====
The ''combo renderer'' is the component that renders each page.


===== Articles Related =====
{{backlinks>.}}

===== Features =====


  * Add Numbering to the [[sectioning|headings]]
  * Add below the first heading:
      * a [[toc|TOC]] 
      * a [[hierarchical_breadcrumb|hierarchical breadcrumb]]
  * Make the [[table|tables responsive]].
  * Overwrite the monospace syntax to an highlight.
  * Enable [[ads:ads|ads inside the content]]

You need to set the [[doku>config:renderer_xhtml|renderer_xhtml]] [[:configuration|configuration]]
