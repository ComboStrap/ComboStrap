====== ComboStrap UI - Icon Component ======


===== About =====
The icon [[ui|component]] permits you to create an icon :
  * from an [[#library|icon library]]
  * or from your media library as a [[#SVG media file]]



===== Articles Related =====
{{backlinks>.}}

===== Syntax =====

<code dw>
<icon name="[library:]name" />
</code>
where:
  * ''name'' is the name of the icon
  * ''library'' is the optional name of the icon library (if not set, the [[#default library|default]] is used)


To show a icon:
  * Go to a [[#library]]
  * Select your icon and remember its name
  * If you are not using the [[#default library|default library]], add the library name as prefix
==== Library ====

We supports the below libraries.

The ''name'' or ''acronym'' can be used as library name.


^ Name ^ Acronym ^ Location ^ 
| ''bootstrap'' | ''bs'' | [[https://icons.getbootstrap.com/|Bootstrap icon]] |
| ''material-design'' | ''md'' | [[https://materialdesignicons.com/|Material design icon]] |
| ''feather'' | ''fe'' | [[https://feathericons.com/|Feather]] |

The default library is ''material-design'' but you can [[#configuration|change this configuration]].

===== Example =====
==== Material Design Icon ====


Example with the [[https://materialdesignicons.com/icon/arrow-down-circle|arrow-down-circle]]

<webcode name="Material-Design">
<code dw>
<icon name="arrow-down-circle" color="danger" width="96px" height="96px" />
</code>
</webcode>


==== Bootstrap ====

Example with the [[https://icons.getbootstrap.com/icons/arrow-down-circle-fill/|arrow-down-circle-fill]]
<webcode name="Boostrap">
<code dw>
<icon name="bootstrap:arrow-down-circle-fill" color="blue" width="48px" height="48px" />
</code>
</webcode>

==== Feather ====
Example with a [[https://feathericons.com/?query=box|''box'' icon]]
<webcode name="Feather">
<code dw>
<icon name="fe:box" color="blue" width="48px" height="48px" />
</code>
</webcode>

==== SVG Media File ====
The ''icon'' component supports the rendering of SVG file located in the [[doku>images|media library]]

By default, ''svg files'' can't be uploaded because they can embedded script and makes it more difficult.

To upload SVG files, you may:
  * transfer the files directly on your server in a location under the media directory (ie ''DOKUWIKI_HOME/data/media'')
  * or allows the upload of SVG file by adding the SVG extension into the [[doku>mime|mime.local.conf file]]
<code properties mime.local.conf>
svg image/svg+xml
</code>


Then, you can render it by giving the media id into the name attributes:
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<icon name="logo.svg" color="orange"/>
</code>
  * Output:
</webcode>

===== Inline SVG =====
This component does not support inline SVG because it's a property of HTML.

If you authorize the direct use of HTML, you can also show inline SVG. See [[icon_inline]]

===== Properties =====

==== Color ====
In the color attribute, you can set a [[styling:color#value|color value]].
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<icon name="logo.svg" color="pink" />
</code>
</webcode>

==== Sizing: Width and Height ====
The ''width'' and ''height'' attributes take the default value of ''24px'' but you can overwrite them.
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<icon name="logo.svg" color="pink" width="96px" height="96px" />
</code>
</webcode>
==== Native Attribute ====
As all [[styling:html|ComboStrap UI components]] do, the ''icon'' component supports also all attributes of its underlying ''svg HTML element''. See [[https://developer.mozilla.org/en-US/docs/Web/SVG/Element/svg|svg attributes]]



===== Configuration =====
==== Icons namespace ====
When the ''icon'' does not exist, the component downloads the icons in the namespace set in the ''icons_namespace'' configuration.

The default value is '':combostrap:icons''

<note type="tip">
On the server, the namespace is a location from the media directory: ''DOKUWIKI_HOME/data/media/''
{{:ui:icon_default_location_combostrap.png|}}
</note>

==== Default Library ====

You can change the default library with the ''defaultIconLibrary'' [[:configuration|configuration]].

At installation time, the default library is [[#material design|material design]].

===== Support =====
==== Prerequisite ====

This component uses the [[support:simplexml|simplexml]] module that is enabled by default. It should be then no problem for most of the platform.
