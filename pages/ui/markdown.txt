====== ComboStrap - Markdown  ======

===== About =====
''ComboStrap'' supports the following ''markdown'' syntax.





^ Syntax ^ Combo Component ^
| [[https://daringfireball.net/projects/markdown/syntax#precode|markdown code]] | [[ui:preformatted|indented code block]] |
| [[https://github.github.com/gfm/#fenced-code-blocks|fenced code block]] | [[ui:code|code]] |
| [[https://github.github.com/gfm/#atx-headings|Atx Heading]] | [[ui:heading#markdown|Atx heading]] |


===== Support =====


Because the markdown syntax was never fully specified, there is a lot of different flavor. We follow in order the following specification:
  * https://daringfireball.net/projects/markdown/syntax
  * https://github.github.com/gfm
  * https://spec.commonmark.org/
