====== ComboStrap UI - tabs ======

===== About =====
Tabs organize content across different screens, data sets, and other interactions.

A ''tab'' is an interactive element that, when activated, displays its associated ''tabpanel''.
===== Example =====

<webcode name="Tabs" frameborder=0 >
<code dw>
<tabs spacing="mb-3">
<tab panel="home">Home</tab>
<tab panel="profile">Profile</tab>
</tabs>
<tabpanels>
<tabpanel id="home">Home</tabpanel>
<tabpanel id="profile">Profile</tabpanel>
</tabpanels>
</code>
</webcode>
