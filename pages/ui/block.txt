====== ComboStrap UI - Block ======


===== About =====
A ''block'' is a [[ui|component]] that wraps/encapsulate some content.

If the component should start at a new line or expand on multiple line and you can draw some border around it, that's a block.

{{backlinks>.}}

===== List of block component =====

  * [[blockquote|Blockquote]]: A block of text with a citation.
  * [[box|Box]] - a generic box for advanced styling
  * [[card|Card (teaser)]]: Do you want to put a text in a box with optionally an image and a button, that's a card
  * [[code|Code]] - a block of code
  * [[console|Console]] - a block representing the output of a command
  * [[file|File]] - a file
  * [[slide|Slide]] - A slide used to create landing page or hero content
  * [[jumbotron|Jumbotron]]: Prominently at the top of your page emphase a content.
  * [[list|List]] - a list
  * [[math|Math]]: Writing math formulas
  * [[note|Note]]: An ancillary text that needs attention.
  * [[related|Related]]: A component showing the backlinks
  * [[webcode|Webcode]] - Output the result of web code (HTML, Javascript, CSS)
  * [[text|Text]] - Create one or more paragraph with the same styling
