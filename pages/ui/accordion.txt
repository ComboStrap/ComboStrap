====== ComboStrap UI - Accordion ======


===== About =====
''accordion'' is a [[ui:ui|UI component]] that layouts [[card|cards]] in an accordion fashion.


<note tip>
You can also choose another layout such as:
  * [[ui:card_column|columns]]
  * [[tabs|tabs]]
</note>

<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<accordion>
    <card collapsed="false">
        <label>Collapsible Group Item #1</label>
        Some placeholder content for the first accordion card. This card is shown by default because of the ''collapsed'' attribute.
    </card>
    <card>
        <label>Collapsible Group Item #2</label>
        === Title ===
        Some placeholder content for the second accordion card. Card are hidden by default.
    </card>
</accordion>
