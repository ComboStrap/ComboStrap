====== ComboStrap UI - Blockquote ======



===== About =====
A ''blockquote'' component is a content with a [[cite|citation]].

===== Articles Related =====
{{backlinks>.}}


===== Syntax =====
A ''blockquote'' has two type:
  * ''typo'' for the [[https://getbootstrap.com/docs/4.0/content/typography/#blockquotes|the typographic Blockquote]]
  * ''card'' (default) for the [[https://getbootstrap.com/docs/4.0/components/card/#header-and-footer|the card Blockquote]]

<code dw>
<blockquote type="card">
=== Header ===
content
<cite>The reference</cite>
</blockquote
</code>

<note tip>
A ''card'' blockquote is just a shortcut that wraps the typographic blockquote in a [[card|card]] 
</note>





===== Example =====

==== Basic ====
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<blockquote>
The true work does not reveal itself until you get into.
<cite>Nico</cite>
</blockquote>
</code>
</webcode>

==== With a class ====
As a [[html|superset HTML element]], you can add your own class or [[style|style]]
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<blockquote class="p-4 m-4 mx-auto" style="border-color:blue;max-width:400px">
The true work does not reveal itself until you get into.
<cite>Nico</cite>
</blockquote>
</code>
</webcode>
==== With a header ====
An header is supported in the ''card'' blockquote.
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<blockquote width="300px" align="center">
=== True work ===
The true work does not reveal itself until you get into.
<cite>Nico</cite>
</blockquote>
</code>
</webcode>
==== In a card ====
You can wrap a typographic blockquote in a card in order to add an image header .
<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<card width="18rem">
{{teaser_image_2.jpg?25|}}
=== True Work ===
<blockquote typo class="mb-0">
The true work does not reveal itself until you get into.
<cite>Nico</cite>
</blockquote>
</card>
</code>
</webcode>

<note tip>
This will output the [[https://getbootstrap.com/docs/4.0/components/card/#header-and-footer|Bootstrap blockquote card of the documentation]]
</note>
==== Card Columns ====

A ''blockquote'' with the type ''card'' can also be wrapped in the [[card_column|card-columns layout component.]]

<webcode name="Default" frameborder=0 width=100% scrolling=yes externalResources="," renderingMode=story >
<code dw>
<card-columns>

<blockquote>
A Blockquote is wrapped in a [[card|card]] by default
<cite>Say the [[card|card]]</cite>
{{::teaser_image_1.png?400}}
</blockquote>

<blockquote>
My Blockquote Without Cite
{{::teaser_image_2.jpg}}
</blockquote>

<blockquote>

</blockquote>


</card-columns>
</code>
</webcode>
