====== ComboStrap UI - Highlighting (mark) ======


===== About =====
''highlight'' is a [[styling:typo|typographic component]] that highlights a sequence of words.


''Highlighting'' permits to add [[styling:emphasis|emphasis]] on a couple of words.

===== Articles Related =====
{{backlinks>.}}

===== Syntax and Example =====
You needs to wrap your words with two quote characters ''%%''%%''.
  * Syntax
<webcode name="Default" frameborder="0">
<code dw>
This text is not highlighted while ''this one is''. 
</code>
  * Output:
</webcode>

===== Configuration =====
==== Enable / Disable ====

To enable the highlight feature, you should choose the [[ui:renderer|Combo renderer]]

The standard dokuwiki syntax calls it [[doku>wiki:syntax#basic_text_formatting|monospaced]] and use the ''code'' html but this syntax replaces it with the ''mark'' element.
