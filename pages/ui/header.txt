====== ComboStrap UI - Header ======


===== About =====
''headers'' define [[wiki:syntax#sectioning|section]] and generates the structure of the document. The table of content (TOC) is created from this structure.


===== Component =====

[[ui|ComboStrap UI]] supports headers   in the following component:
   * [[card]]
   * [[blockquote]]
   * [[note]]

<note info>This headers are local and does not modify the structure of the document</note>
===== Numbering =====

With the [[rplus|rplus renderer]], you can number them.
