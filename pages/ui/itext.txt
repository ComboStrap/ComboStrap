====== ComboStrap UI - itext ======


===== About =====
''itext'' is a [[ui:typo|typographic component]] that permits to applies [[styling:typo|typographic attribute]] to one or more words.

<note tip>
If you want to applies the [[styling:typo|typographic attributes]] on a paragraph, you should use the [[text|text component]].
</note>


===== Syntax =====


<code dw>
<itext ></itext>
</code>
