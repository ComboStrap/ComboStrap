====== ComboStrap Quality - Dynamic Page Monitoring ======


===== About =====
''Dynamic Quality Page Monitoring'' is a feature that encourages the ''writer'' (users with edit permission) to improve the [[score|quality score]] of its page. 

A message is shown that indicates the [[rule|quality rules]] that can be improved and how.

You can exclude [[rule|quality rules]] from ''Dynamic Monitoring'' via [[#configuration]].

===== Configuration =====
==== Excluded Quality Rules ====
The [[:configuration|configuration]] ''qualityExcludedMonitoredQualityRules'' lists the [[rule|quality rules]] that should not be monitored.
