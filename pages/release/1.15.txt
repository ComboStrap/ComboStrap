====== Release 1.15 ======

===== About =====
The release ''1.15'' is a quick bug fix release above the [[1.14|1.14]].



===== Bug Resolved =====
  * Image Resize by height has bad height and with ratio in the ''srcset'' url when resizing by height.
  * [[ui:outline|Outline numbering]] didn't take into account the [[doku>config:toptoclevel]]
  * ''unformatted'' has an unknown display
  * ''webcode'' html output was escaped two times making it to print HTML and not rendering it.
