====== Strap Template - Header Slot ======


===== About =====
The ''header slot'' is a [[slot|slot]] that contains the ''header'' of the page.

The ''page header'' is rendered at the top of the page and contains generally a [[ui:menubar|menu bar]]. 




<note>
  * The ''how-to'' getting started guide explains how to create a header navigational bar: [[getting_started:5_menu]].
  * It defines a header for the page. If you want to define a header for a component, see the [[ui:header|header component]]
</note>
