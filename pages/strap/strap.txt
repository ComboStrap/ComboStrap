====== Strap - Template ======

===== About =====
''strap'' is the template part of [[:start|ComboStrap]]

===== Installation and configuration =====

<btn class="mb-5">
See [[getting_started:getting_started|the getting started guide]]
</btn>





===== Features =====
  * [[bootstrap|Last bootstrap version (4)]] or [[bootstrap_custom|create and add your own]]
  * The layout supports [[bar|3 bars]]
      * [[doku>faq:sidebar|Sidebar]]
      * [[headerbor]]
      * [[footerbor]]
  * [[dynamic_grid|Dynamic grid system from 12 to 16 columns]]
  * [[getting_started:4_favicon|Icon]]: Recognized automatically the [[getting_started:4_favicon|icon]] ''android-chrome-192x192.png'' generated by https://realfavicongenerator.net/
  * Support [[tagline|tagline]]
  * Support the [[doku>conf:youarehere|youarehere]] configuration for a [[ui:breadcrumb#trailing|trailing breadcrumb]]
  * [[:ui:scale|Size Scale configuration (font and spacing)]]
