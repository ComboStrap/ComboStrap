====== Automatic Hidden Page ======


===== About =====

The [[:combo|combo plugin]] automatically [[doku>config:hidepages|hides]]:
  * the [[bar|bars]]
  * the ''combostrap'' private namespace.


This page are still accessible but should not be advertised to the outside world because they are just component and not fully readable page.

This is a component of the [[:seo|seo module]].

===== Explanation =====


This functionality adds automatically to the [[doku>config:hidepages]] [[:configuration|configuration]], the following pattern 
<file>
(headerbar|footerbar|sidekickbar|sidebar|combostrap)
</file>
