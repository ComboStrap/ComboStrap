====== ComboStrap Metadata - Website (Home Page) ======


===== About =====
A ''WebSite'' page is a page with [[semantic|metadata]] about your website. 

''ComboStrap'' sets automatically the home page with the ''website'' [[type|type]] and implements the [[https://developers.google.com/search/docs/data-types/sitelinks-searchbox|searchbox]] [[google|google rich result]] 


<note>
A website is defined as a set of related web pages that are served from a single web domain.
</note>





===== Articles Related =====
{{backlinks>.}}


===== List =====
The following elements are added to the home page:
  * ''url''
  * ''name'', the [[doku>config:title|title]]
  * ''description'', the [[semantic:tagline|tagline]]
  * ''image'', the [[getting_started:4_favicon|logo]] (as png)


<note>
You can set more metadata yourself, check the example and what kind of properties may be added at [[https://schema.org/WebSite|Schema.org WebSite]]
and go to the [[google|Google Rich Result page]] to know how.
</note>
