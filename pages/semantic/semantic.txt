====== ComboStrap - Page Metadata (Semantic) ======


===== About =====
The ''metadata manager'' is composed of the [[frontmatter|frontmatter module]] that permits to set the following web structured data.

The metadatas can be seen by default thanks to the [[viewer|Metadata viewer]].

===== Usage =====

They below internal metadata are used by ''ComboStrap'':
      * [[canonical|canonical]]
      * [[description|description]]
      * [[title|title]]
      * [[published|published date]]
      * [[type|page Type]]
      * [[image|image]]

and enables what we called a ''semantic component''.

==== Internal ====

They allows internally:

  * [[url:redirection|URL redirection processing]]
  * [[automatic_canonical|Automatic Canonical]]
  * [[url:permalink|Permalink]]
  * [[quality:lpp|Late Publication Protection]]
