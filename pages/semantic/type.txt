====== ComboStrap Page Metadata - Type ======


===== About =====
''type'' is a [[semantic|metadata]] that define the type of a page. This information is used by the [[facebook|facebook social graph]].

<badge>Since [[:changes#version_112|version 1.12]]</badge>

===== How to set the value =====
You can set the value in the [[frontmatter|frontmatter]]

<code json>
{
    "type":"website"
}
</code>
 
The value can be one of the following:

^ Value ^ Facebook (og) ^ Google (ld-json) ^ Description ^
| [[website|website]] (default) | ''website'' (default) | ''website'' | General website information |
| [[article|article]] | ''article'' | ''article'' | A published page. The content will not change after publication - blog, newspaper,.. |
| ''book'' | ''book'' | [[https://developers.google.com/search/docs/data-types/book|book]] | A book |


Others Og:
  * ''music''
  * ''video''
  * ''profile'' 

Ld-data:
  * NewsArticle
  * Article
  * TechArticle
  * BlogPosting
  * Recipe
